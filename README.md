# CSS Externo

Projeto desenvolvido com foco na separação entre estrutura e estilo, utilizando CSS externo para a estilização da página.

## Descrição
Este projeto consiste em uma página web estática construída em HTML, onde toda a estilização é feita em um arquivo CSS separado, importado no HTML através da tag `<link>`.

O objetivo foi aplicar boas práticas de organização do código, facilitando a leitura, manutenção e reutilização dos estilos.

## Tecnologias utilizadas
- HTML5  
- CSS3 (CSS externo)  

## Objetivo
- Praticar a estruturação semântica do HTML  
- Aplicar estilização usando CSS externo  
- Melhorar a organização e manutenção do código  
- Compreender a separação de responsabilidades no desenvolvimento web  

## Observações
Este projeto foi desenvolvido para fins de estudo.  
O uso de CSS externo é recomendado em projetos maiores e profissionais por facilitar a escalabilidade do código.
